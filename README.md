# Ten Ten Seafood & Dim Sum — Premium Website

A modern, premium restaurant website for Ten Ten Seafood & Dim Sum Restaurant built with the **P.A.G.E.S framework** (Purpose, Animation, Great Copy, Elevate, Ship).

## Tech Stack

- **Next.js 15** (App Router)
- **Tailwind CSS 4**
- **Framer Motion**
- **Lucide Icons**
- **TypeScript**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment (Vercel)

1. Push to a GitHub repository
2. Import in [Vercel](https://vercel.com)
3. Deploy — Vercel auto-detects Next.js
4. Add custom domain when ready

## Sections

| Section | Description |
|---------|-------------|
| **Hero** | Cinematic parallax hero with animated entrance, floating particles, and dual CTAs |
| **About** | Brand story with image, stats strip, and mini-features |
| **Food Features** | Three alternating image+text blocks highlighting cuisine categories |
| **Menu** | Tabbed menu with 4 categories, real prices, and bilingual names |
| **Gallery** | 8-image grid with hover effects and keyboard-accessible lightbox |
| **Testimonials** | Horizontal scrolling review cards with 5-star ratings |
| **Contact** | Google Maps embed, contact cards, and styled contact form |
| **Footer** | Final CTA strip, quick links, and contact info |
